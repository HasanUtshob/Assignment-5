Question-1: What are the different ways to select an element in the DOM?
৬টি উপায়:
০১. document.getElementById()
০২. document.getElementsByClassName()
০৩. document.querySelector()
০৪. document.querySelectorAll()
০৫. document.getElementsByTagName()
০৬. document.getElementsByName()
Question-2: What is the difference between innerHTML, innerText, and textContent ?
০১. innerHTML :- কোনো এলিমেন্টের HTML কনটেন্ট পেতে বা সেট করতে।
০২. innerText :- শুধু ভিজিবল টেক্সট কনটেন্ট পেতে বা সেট করতে।
০৩. textContent :- সব টেক্সট (hidden সহ) পেতে বা সেট করতে।
Question-3: What is event delegation in the DOM?
Event delegation হল এমন একটি টেকনিক যেখানে parent এলিমেন্ট event handle করে child গুলোর জন্য।
Question-4: What is event bubbling in the DOM?
Event bubbling হচ্ছে ইভেন্ট শুরু হয় টার্গেট এলিমেন্ট থেকে এবং উপরের দিকে propagate হয়।
Question-5: How do you create, add, and remove elements using JavaScript?
০১. Create Element : document.createElement();
০২. যোগ করা: parent.appendChild(element)
০৩. Remove: element.remove() বা parent.removeChild(element)
